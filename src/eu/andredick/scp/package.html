SCP-Komponente (Definition des Set Covering Problem)

Das Set Covering Problem ist im Kontext der ACO-Metaheuristik eine Ausprägung des abstrakten Problems {@link eu.andredick.aco.problem}.

Komponenten:

  • SCProblem (Klasse der Set Covering Probleme) {@link eu.andredick.scp.SCProblem}
  • SCPSolution (Lösung des Set Covering Problems) {@link eu.andredick.scp.SCPSolution}
  • Structure (Strukturmatrix eines Set Covering Problems) {@link eu.andredick.scp.Structure}
  • ObjectiveFunction (Zielfunktion des Set Covering Problems) {@link eu.andredick.scp.ObjectiveFunction}
