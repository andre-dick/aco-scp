014 * Anschließend benutzt {@link AbstractPheromoneAssociation} eine Ausprägung der Komponente Pheromon-Initialisierung zur Anpassung aller Pheromon-Konzentrationen.<br>
015 * <br>
016 * <p><img src="{@docRoot}/images/PheromoneInit.svg" alt=""></p>
017 */
