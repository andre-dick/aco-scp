003/**
004 * <b>Abstrakte Klasse von Problemen</b>, welche durch die ACO-Metaheuristik zu lösen sind<br>
005 * <br>
006 * Diese Abstrakte Komponente wird auf der generischen Ebene der ACO-Metaheuristik verwendet,<br>
007 * auf welcher noch keine Abhängigkeiten zu einer bestimmten Problem-Klasse bestehen.<br>
008 * Genauso können auch spezialisierte Komponenten der ACO-Metaheuristik das Abstrakte Problem referenzieren, wenn sie von keiner bestimmten Problem-Klasse abhängig sind.<br>
009 * <br>
010 * Um eine Problem-Klasse zu definieren, muss die Abstraktes Problem und die zugehörige Abstakte Lösung abgeleitet werden.<br>
011 *
012 * <p><img src="{@docRoot}/images/Problem.svg" alt=""></p>
013 */
014public abstract class AbstractProblem {
015
016    /**
017     * Liefert den Namen der Instanz eines Problems
018     *
019     * @return Name des Problems
020     */
021    public abstract String getName();
022}
